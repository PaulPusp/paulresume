GRACE Drought and Terrestrial Water Storage Dynamics

This project uses GRACE/GRACE-FO and climate datasets to analyze terrestrial-water-storage behavior, drought response, and hydroclimate patterns using descriptive analysis and machine-learning classification logic.

Keywords

GRACE, GRACE-FO, terrestrial water storage, drought analysis, SPEI, hydroclimate, Bangladesh, climate data analysis, machine learning classification.

Methods and tools

  • Processing of GRACE/GRACE-FO storage anomaly datasets.
  • Integration of climate and drought indicators.
  • Pattern interpretation for storage-drought response.
