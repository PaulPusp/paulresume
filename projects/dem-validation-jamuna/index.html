DEM Validation for Jamuna Floodplain Hydrology

This project focuses on evaluating open-access DEMs for floodplain analysis and flood-depth modelling support in low-relief terrain. It uses water- and land-domain reference data, error metrics, and uncertainty-aware interpretation.

Keywords

DEM validation, Jamuna floodplain, flood-depth mapping, hydrological suitability, Copernicus GLO-30, NASADEM, SRTM, ALOS AW3D30, GIS, remote sensing.

Methods and tools

  • Surface-dependent DEM assessment over water and land.
  • Error metrics and uncertainty analysis for low-relief floodplain terrain.
  • Decision-oriented DEM ranking for hydrological and flood-depth applications.
