Sentinel-1 SAR Flood Mapping and Depth Estimation in Bangladesh

This project presents a geospatial AI workflow for flood information in Bangladesh, combining Sentinel-1 SAR imagery, deep-learning flood segmentation, DEM assessment, FwDET flood-depth estimation, and GIS-based impact interpretation.

Keywords

Sentinel-1 SAR flood mapping, Attention U-Net, DeepLabV3+, FwDET flood depth, DEM validation, Bangladesh floodplain, ArcGIS Pro, Google Earth Engine, Python geospatial automation.

Methods and tools

  • Sentinel-1 before/after SAR feature-stack preparation.
  • Flood-extent segmentation with deep learning workflows.
  • DEM-supported flood-depth estimation using FwDET logic.
  • Parcel, building, and local-scale GIS interpretation for decision support.
