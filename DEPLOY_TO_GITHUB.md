# Deploy this portfolio to GitHub Pages

Repository target: `PaulPusp/paulresume`

## Fast upload method

1. Open the repository on GitHub.
2. Upload/replace these files at the repository root:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `.nojekyll`
   - `assets/site-preview.svg`
   - `data/resume.json`
3. Commit directly to the `main` branch.

## Enable GitHub Pages

Go to:

`Settings` → `Pages` → `Build and deployment`

Set:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

Expected public URL after GitHub Pages is enabled:

`https://paulpusp.github.io/paulresume/`

## Privacy note

The public website uses email, GitHub, and LinkedIn only. Phone/WhatsApp numbers were intentionally not exposed to avoid spam scraping.
